Comenzar

¿Qué es HTML5?

HTML5 es la versión más reciente del lenguaje de marcado HTML, utilizado para crear y estructurar páginas web. Permite organizar contenido como textos, imágenes, videos y formularios de manera sencilla y eficiente. Es fundamental en el desarrollo web moderno porque es compatible con la mayoría de los navegadores y dispositivos.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

La estructura básica de HTML5 está compuesta por etiquetas principales como <!DOCTYPE html>, <html>, <head> y <body>. El DOCTYPE indica que se trata de un documento HTML5, la etiqueta html encierra todo el contenido, head contiene información como el título y configuraciones, y body contiene todo lo visible en la página.

Etiquetas principales

h1 - h6

Las etiquetas h1 a h6 se utilizan para definir títulos y subtítulos en una página web. h1 es el título más importante y h6 el menos importante. Ayudan a organizar la información y mejorar la lectura del contenido.

p

La etiqueta p se utiliza para escribir párrafos de texto. Es una de las más utilizadas en HTML ya que permite mostrar información de forma ordenada.

img

La etiqueta img se usa para insertar imágenes en una página web. Utiliza atributos como src para indicar la ruta de la imagen y alt para describirla.

a

La etiqueta a se utiliza para crear enlaces en una página web. Utiliza el atributo href para especificar la dirección a la que apunta el enlace.

button

La etiqueta button se utiliza para crear botones interactivos. Se pueden usar en formularios o para ejecutar acciones con JavaScript.

ul / ol / li

Estas etiquetas se usan para crear listas. ul crea listas desordenadas, ol listas ordenadas y li define cada elemento de la lista.

HTML5 semántico

header

La etiqueta header representa el encabezado de la página o de una sección. Generalmente contiene el logo, el título y el menú de navegación.

nav

La etiqueta nav representa la navegación de la página. Contiene enlaces a otras páginas o secciones de la misma página.

main

La etiqueta main contiene el contenido principal de la página. Solo debe existir una vez por documento.

section

La etiqueta section se utiliza para dividir el contenido en secciones temáticas.

article

La etiqueta article representa contenido independiente como artículos o publicaciones.

footer

La etiqueta footer representa el pie de página, donde se incluye información como derechos de autor o datos adicionales.

Formularios en HTML5

Los formularios en HTML5 se utilizan para recopilar datos del usuario dentro de una página web. Se crean con la etiqueta form, que funciona como contenedor principal. Dentro de ella se utilizan etiquetas como label para indicar qué información se debe ingresar, input para capturar datos en diferentes formatos como texto o correo, textarea para escribir mensajes largos, select para crear listas desplegables junto con option que define cada opción, y button que permite enviar la información o ejecutar una acción. Además, HTML5 incluye etiquetas importantes como h1 a h6 para títulos, p para párrafos, img para imágenes, a para enlaces, y ul, ol y li para listas. También existen etiquetas semánticas como header, nav, main, section, article y footer, que ayudan a organizar mejor la estructura del contenido. Por último, HTML5 permite integrar multimedia mediante etiquetas como audio y video, haciendo las páginas más dinámicas e interactivas.

Multimedia en HTML5

La multimedia en HTML5 permite agregar contenido como imágenes, audio y video dentro de una página web, haciendo que sea más interactiva y atractiva.

Imágenes

La etiqueta img se utiliza para mostrar imágenes en la página web.

Audio

La etiqueta audio permite reproducir sonidos o música directamente en la página.

Video

La etiqueta video permite mostrar y reproducir videos dentro del navegador.

Accesibilidad

El atributo alt se usa en las imágenes para describir su contenido. Es importante para la accesibilidad y para personas con discapacidad visual.

Conclusión

HTML5 es una herramienta muy importante para el desarrollo web, ya que permite crear páginas organizadas, interactivas y fáciles de usar. Además, facilita la integración de formularios y contenido multimedia.
